# SIGED-Lampa Frontend (Fase 6A)

Aplicacion React + TypeScript para la estructura navegable de SIGED-Lampa. La configuracion QA usa `VITE_DATA_MODE=real`; los estados `hybrid` y `mock` estan catalogados para no presentar datos demostrativos como datos reales.

## Scripts

- `npm run dev`
- `npm run lint`
- `npm test`
- `npm run test:coverage`
- `npm run build`

El backend sigue siendo la autoridad de identidad, roles, permisos y propiedad de los recursos.
